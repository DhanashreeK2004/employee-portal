# Employee Portal

A simple Employee Portal built using Angular.

## Features

- Login Page
- Role-Based Access
- Dashboard
- Add User
- Edit User
- Delete User
- Search User
- Responsive UI

## Technologies Used

- Angular
- HTML
- CSS
- TypeScript
